# Mount Everest

Welcome to the Mount Everest information website! This project provides facts and information about the world's highest mountain.

## Features

- 📍 Located in the Himalayas on the Nepal-Tibet border
- 🏔️ Height: 29,032 feet (8,849 meters)
- 🎯 First ascent: May 29, 1953
- 🌍 Home to an interactive informational website

## About This Site

This is a simple HTML-based website with information about Mount Everest, including:
- Basic facts and statistics
- Historical information about the first ascent
- Details about climbing routes

## Getting Started

Simply open `index.html` in a web browser to view the site.
